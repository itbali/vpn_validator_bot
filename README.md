# VPN Validator Bot

Telegram бот для управления VPN-сервером на базе Outline VPN.

## Основные функции

- Автоматическая проверка членства в канале
- Управление VPN-ключами (создание, удаление, перевыпуск)
- Мониторинг состояния сервера
- Сбор и анализ статистики использования
- Система уведомлений о проблемах

## Новые функции

### Статистика использования
- Графики использования трафика за день/неделю/месяц
- Анализ пиковых часов нагрузки
- Расчет среднего времени использования VPN
- Общая статистика по всем пользователям (для админов)

### Мониторинг сервера
- Отслеживание загрузки CPU, RAM и диска
- Мониторинг количества подключений
- Автоматические уведомления при превышении пороговых значений
- Проверка доступности VPN-сервера

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/vpn_validator_bot.git
cd vpn_validator_bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл .env на основе .env.example и заполните необходимые параметры:
```bash
cp .env.example .env
nano .env
```

## Настройка

### Переменные окружения

- `BOT_TOKEN` - токен Telegram бота
- `CHANNEL_ID` - ID канала для проверки подписки
- `ADMIN_IDS` - список ID администраторов (через запятую)
- `OUTLINE_API_URL` - URL API Outline VPN
- `OUTLINE_CERT_SHA256` - SHA256 сертификата сервера
- `CHECK_MEMBERSHIP_INTERVAL` - интервал проверки членства (в секундах)
- `SERVER_CHECK_INTERVAL` - интервал проверки сервера (в секундах)
- `CPU_THRESHOLD` - порог загрузки CPU (в процентах)
- `RAM_THRESHOLD` - порог загрузки RAM (в процентах)
- `DISK_THRESHOLD` - порог загрузки диска (в процентах)
- `TRAFFIC_THRESHOLD` - порог использования трафика (в процентах)

## Запуск

```bash
python main.py
```

## Команды бота

### Пользовательские команды
- `/start` - начать работу с ботом
- `/help` - показать справку
- `/status` - проверить статус VPN
- `/stats` - показать статистику использования
- `/regenerate` - перевыпустить ключ
- `/delete` - удалить ключ
- `/faq` - часто задаваемые вопросы
- `/support` - связаться с поддержкой

### Админские команды
- Просмотр общей статистики
- Анализ пиковых часов
- Мониторинг состояния сервера
- Управление пользователями

## Логирование

Логи сохраняются в файл `/opt/vpn_bot/bot.log`. Уровень логирования можно настроить в файле `config/logging.py`.

## Мониторинг

Бот автоматически отслеживает:
- Загрузку системных ресурсов
- Количество активных подключений
- Использование трафика
- Доступность VPN-сервера

При превышении пороговых значений администраторы получают уведомления.

## Статистика

Для просмотра статистики используйте команду `/stats`. Доступны следующие данные:
- График использования трафика
- Среднее время использования
- Пиковые часы нагрузки
- Общая статистика (для админов)
